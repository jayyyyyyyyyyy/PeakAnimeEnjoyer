# Notifications

## Status

Implemented as in-app derived notifications.

No notification table is required yet.

---

## Implemented Events

Season Started

Shown when a season is ACTIVE.

Review Available

Shown when a season is in REVIEW and the current member has not submitted a review.

Reveal Ready

Shown to the owner when every member has submitted a review.
